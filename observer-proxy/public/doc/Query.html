Class: Query

Query

new Query(uri, type)

The Query object is used to represent and manipulate SPARQL queries using the JavaScript Object Notation (JSON). The prototype function provides a number of useful functions for manipulating queries as well as converting them between different formats (specifically SPARQL and SPIN).

Parameters:
Name Type Description
uri

URI identifying the new Query object
type

The type of query modeled; see also QueryType
