Class: QueryCanvas

App. QueryCanvas

QueryCanvas provides the core model for the application by maintaining a collection of query instances cloned from external SPARQL endpoints. It provides operations to manipulate those queries and will notify other components in the system of changes to the internal model using the jQuery event triggering mechanism.
