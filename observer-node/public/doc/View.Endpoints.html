Class: Endpoints

View. Endpoints

View.Endpoints provides functionality to add, edit, and remove endpoints to Collustra's model. It will also configure HTML5 drag and drop so that users can drag links to external SPARQL endpoints directly into the application to bootstrap the discovery and construction process.
